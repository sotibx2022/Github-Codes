Git Commands

Local Code Editor Git Commands GitHub Git Commands
Initialize a Git Repository (git init) Clone a Repository (git clone <repository_url>)
Add Changes to Staging Area (git add <file_name>) Push Changes to Remote Repository (git push <remote_name> <branch_name>)
Commit Changes (git commit -m "Commit message") Fetch Changes from Remote Repository (git fetch <remote_name>)
Check Status of Repository (git status) Pull Changes from Remote Repository (git pull <remote_name> <branch_name>)
View Commit History (git log) Delete a Remote Branch (git push <remote_name> --delete <branch_name>)
Create a New Branch (git branch <branch_name>) View Remote Repositories (git remote -v)
Switch Branches (git checkout <branch_name>)
Merge Branches (git merge <branch_name>)
Delete a Branch (git branch -d <branch_name>)
Undo Changes in Working Directory (git checkout -- <file_name>)
Undo Last Commit (Keep Changes) (git reset HEAD^)
Undo Last Commit (Discard Changes) (git reset --hard HEAD^)
